# Group One - web optimizada para GitHub Pages

Archivos listos para subir a GitHub Pages.

## Editar antes de publicar
- Reemplazar `+54 11 XXXX-XXXX` por el teléfono real.
- Confirmar el mail `hola@groupone.com.ar`.
- Cambiar las URLs de `robots.txt` y `sitemap.xml` por la URL definitiva del repositorio o dominio.
- Si querés usar imágenes reales, reemplazá los SVG dentro de `assets/` manteniendo los mismos nombres.

## Publicación
Subí todos los archivos a la raíz del repo y activá GitHub Pages desde Settings → Pages.
